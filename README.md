# OTOÑO app

App e-commerce de venta de ropa, hecho con el propósito de poner en práctica y mejorar habilidades con Ruby on Rails.
Se usarán gemas como:
- Devise
- JWT
- CANCANCAN
- I18n
- Annotate
- ViewComponent
- Stimulus
- Turbo
- Stimulus Reflex
- Sidekiq

El webpacker elegido es esbuild.

## Ruby on Rails 7.1.3.2
## Ruby 3.2.3
